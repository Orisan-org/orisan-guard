# Guard Extension Scaffold

This is a Manifest V3 scaffold for Orisan Guard.

Current status:

- no prompt interception
- no content scripts
- no host permissions
- no network calls
- no cloud sync
- no background collection
- no Chrome Web Store release

The extension shell exists only to reserve the browser-extension shape while the local Guard core matures.

Future extension work must keep raw prompts local unless a later task explicitly changes the privacy model.
